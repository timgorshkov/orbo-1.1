
Описание файлов для производственной среды:

"ИНН_ИД".crt, "ИНН_ИД".key - Клиентский сертификат и его секретная часть сгенерированная на основе client_ca.crt (пароль 1234)
client_ca.crt - публичная часть ключа которым подписан клиентский сертификат.
rsa_2048_private_key.xml - приватный ключ организации для подписи запросов.
rsa_2048_public_key.xml  - публичный ключ организации для подписи запросов.
Имя приватного ключа - ИНН_ИД организации.(ИД - фигурирует в названии файлов и архива ИНН_ИД)
