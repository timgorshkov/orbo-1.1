# Заявки (Applications) — обзор

## Зачем это нужно

Раздел «Заявки» — универсальный инструмент для приёма и обработки заявок от участников сообщества. Подходит для:

- **Заявок на вступление** — закрытые группы, бизнес-клубы, профессиональные сообщества
- **Заявок на услуги** — консультации, курсы, партнёрские предложения от автора канала
- **Обработки join-request** — автоматический приём заявок в Telegram-группу

Заявки связаны с участниками (как контакты в CRM) и проходят по настраиваемой воронке статусов.

---

## Ключевые понятия

| Термин | Описание |
|--------|----------|
| **Воронка (Pipeline)** | Набор этапов, через которые проходит заявка |
| **Этап (Stage)** | Конкретный статус в воронке (например, «Новая», «На рассмотрении», «Одобрена») |
| **Форма заявки** | Что видит пользователь: лендинг + анкета + страница успеха |
| **MiniApp** | Форма заявки, доступная прямо в Telegram |
| **Spam-score** | Автоматическая оценка, насколько профиль похож на бота/спамера |

---

## Как это работает

### 1. Создайте воронку

Определите этапы, через которые проходит заявка. Orbo создаёт набор этапов по умолчанию, которые можно настроить.

### 2. Создайте форму заявки

Форма привязана к воронке и определяет:
- Лендинг — описание сообщества/услуги
- Анкету — какие поля заполняет заявитель
- Страницу успеха — что видит человек после подачи заявки

### 3. Поделитесь ссылкой

Ссылка на MiniApp формата `t.me/orbo_event_bot?startapp=f-{slug}` работает прямо в Telegram. Также доступна обычная веб-ссылка.

### 4. Обрабатывайте заявки

Заявки появляются в разделе «Заявки» с автоматическим spam-score. Перемещайте их по этапам воронки, одобряйте или отклоняйте.

---

## Типы воронок

| Тип | Назначение |
|-----|-----------|
| `join_request` | Заявки на вступление в группу/сообщество |
| `service` | Заявки на услугу или продукт |
| `custom` | Произвольная воронка |

---

## UTM-метки

Ссылки на формы заявок совместимы с UTM-параметрами. При переходе по ссылке с UTM-метками данные сохраняются в источнике заявки. Это позволяет в будущем анализировать эффективность каналов привлечения.
