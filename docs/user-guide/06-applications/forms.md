# Формы заявок и MiniApp

## Что такое форма заявки

Форма — это то, что видит заявитель. Каждая форма привязана к воронке и состоит из трёх частей:

1. **Лендинг** — описание сообщества или услуги
2. **Анкета** — поля, которые заполняет заявитель
3. **Страница успеха** — сообщение после подачи заявки

---

## Создание формы

### Шаг 1. Откройте воронку

Перейдите в раздел «Заявки» → выберите воронку.

### Шаг 2. Нажмите «Создать форму»

### Шаг 3. Настройте лендинг

| Поле | Описание |
|------|----------|
| Заголовок | Название, которое увидит заявитель |
| Описание | Текст о сообществе или условиях |
| Изображение | Обложка (опционально) |

### Шаг 4. Настройте анкету

Определите, какие данные собирать:

| Поле | Примеры |
|------|---------|
| Имя | Имя и фамилия |
| Контакт | Telegram, email, телефон |
| О себе | Профессия, компания |
| Произвольные поля | Любые вопросы, которые нужны |

### Шаг 5. Настройте страницу успеха

Что увидит заявитель после подачи заявки: благодарность, инструкции, ссылки.

### Шаг 6. Сохраните

---

## Ссылки на форму

После создания формы доступны два типа ссылок:

### MiniApp-ссылка (Telegram)

```
t.me/orbo_event_bot?startapp=f-{slug}
```

Открывается прямо в Telegram. Данные пользователя из Telegram подставляются автоматически.

### Веб-ссылка

```
https://my.orbo.ru/e/{slug}
```

Для использования вне Telegram: email-рассылки, сайт, соцсети.

---

## Как использовать ссылку

- Опубликуйте в Telegram-канале или группе
- Добавьте в описание группы или канала
- Используйте в рекламных кампаниях (поддерживаются UTM-метки)
- Отправьте в личные сообщения

---

## Spam-score

Каждая заявка автоматически получает spam-score — оценку, насколько профиль заявителя похож на бота или спамера. Оценка основана на данных из Telegram-профиля (наличие фото, имени, username и т.д.).

Spam-score помогает быстрее отсеивать подозрительные заявки без ручной проверки каждой.

---

## Редактирование и удаление

- Откройте форму из списка в воронке
- Измените нужные настройки (лендинг, анкету, страницу успеха)
- Сохраните изменения

Для удаления формы используйте соответствующую кнопку в настройках.
