# Неотвеченные вопросы

## Зачем это нужно

Получайте уведомление, когда вопрос в группе остаётся без ответа:

- Не пропускайте важные вопросы клиентов
- Контролируйте время ответа (SLA)
- Распределяйте нагрузку в команде

---

## Как это работает

1. AI анализирует сообщения и определяет вопросы
2. Отслеживает, был ли дан ответ
3. Если ответа нет X часов — уведомление
4. Учитывает рабочие часы (опционально)

---

## Настройка правила

### Шаг 1. Создайте правило

Перейдите в **Настройки → Правила уведомлений → Создать**.

### Шаг 2. Выберите тип

Выберите **"Неотвеченный вопрос"**.

### Шаг 3. Настройте параметры

| Параметр | Описание | Пример |
|----------|----------|--------|
| Группы | Какие чаты мониторить | Клиентские чаты |
| Порог | Часы без ответа | 2 часа |
| Рабочие часы | Когда проверять | 09:00 — 18:00 |
| AI-анализ | Включить | ✅ |
| Получатели | Кто получит | Менеджеры |

[СКРИНШОТ: Форма настройки правила неотвеченного вопроса]

### Шаг 4. Сохраните

## Рабочие часы

### Зачем нужны

Если вопрос задан в 23:00, нет смысла получать уведомление в 01:00. Настройте рабочие часы — проверка будет только в это время.

### Как работает

| Время вопроса | Рабочие часы | Когда уведомление |
|---------------|--------------|-------------------|
| 10:00 | 09:00-18:00 | 12:00 (через 2ч) |
| 20:00 | 09:00-18:00 | 11:00 след. дня |
| 08:00 | 09:00-18:00 | 11:00 (с 09:00 + 2ч) |

### Настройка

1. В правиле включите **"Рабочие часы"**
2. Укажите начало и конец (по часовому поясу организации)

---

## Что считается вопросом

AI определяет вопросы по признакам:

- Вопросительный знак
- Вопросительные слова ("как", "когда", "почему")
- Просьбы о помощи
- Запросы информации

**Примеры:**
- "Когда будет готов отчёт?"
- "Подскажите, как оформить заказ"
- "Нужна помощь с настройкой"

---

## Что считается ответом

AI определяет, что на вопрос ответили:

- Прямой ответ на вопрос
- Обращение к автору вопроса
- Информация по теме вопроса

**Не считается ответом:**
- Сообщения от того же автора
- Несвязанные сообщения
- Реакции и эмодзи

---

## Пример уведомления

**В Orbo:**

```
❓ Неотвеченный вопрос в «Чат с ООО Ромашка»

"Когда будет готов отчёт за декабрь?"
— @elena_client, 2 часа назад

Рабочее время: 09:00–18:00

[Открыть в Telegram] [Решено]
```

**В Telegram:**

```
❓ Неотвеченный вопрос в «Чат с ООО Ромашка»

"Когда будет готов отчёт за декабрь?"
— @elena_client, 2 часа назад

[Открыть сообщение]
```

## Рекомендации по настройке

### Для клиентских чатов

| Параметр | Значение |
|----------|----------|
| Порог | 2 часа |
| Рабочие часы | 09:00–18:00 |
| Получатели | Менеджеры проекта |

### Для внутренних чатов

| Параметр | Значение |
|----------|----------|
| Порог | 4 часа |
| Рабочие часы | Можно отключить |
| Получатели | Руководитель команды |

### Для сообществ

| Параметр | Значение |
|----------|----------|
| Порог | 24 часа |
| Рабочие часы | Отключить |
| Получатели | Модераторы |
