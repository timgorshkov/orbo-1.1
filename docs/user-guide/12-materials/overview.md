# База знаний (Материалы)

## Что это такое

Раздел **«Материалы»** — это внутренняя база знаний вашего сообщества. Здесь можно хранить документы, инструкции, полезные ссылки и файлы. Структура организована как дерево папок — как в облачном хранилище.

Материалы доступны:
- Администраторам через панель управления Orbo
- Участникам через публичный портал (если открыт доступ)

---

## Типы контента

| Тип | Описание |
|-----|----------|
| 📄 **Документ** | Текст с форматированием, заголовками, таблицами, изображениями и видео |
| 📎 **Файл** | Загружаемый файл (PDF, Word, архив и т.д.) |
| 🔗 **Ссылка** | Ссылка на внешний ресурс с названием и описанием |
| 📁 **Папка** | Контейнер для группировки материалов |

---

## Структура

Материалы организованы в дерево папок. Вы можете создавать любую вложенность:

```
📁 Онбординг
  📄 Правила сообщества
  🔗 Чат в Telegram
  📁 Обучение
    📄 Модуль 1
    📄 Модуль 2
📁 Ресурсы
  📎 Шаблон договора.pdf
  🔗 Полезные инструменты
```

---

## Просмотр базы знаний

1. В левом меню нажмите **«Материалы»** (📚)
2. Слева — дерево папок, справа — содержимое выбранной папки или документа
3. Кликните на папку — откроется её содержимое
4. Кликните на документ — откроется редактор
