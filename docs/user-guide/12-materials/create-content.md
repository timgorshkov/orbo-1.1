# Создание и редактирование контента

## Создание папки

1. В разделе **«Материалы»** нажмите **«+ Создать»** → **«Папку»**
2. Введите название папки
3. Нажмите **«Создать»**

Папку можно создать внутри другой папки — просто предварительно откройте нужную.

---

## Создание документа

### Шаг 1. Создайте документ

Нажмите **«+ Создать»** → **«Документ»**. Откроется редактор.

### Шаг 2. Напишите и отформатируйте текст

Редактор документов поддерживает:

| Элемент | Как добавить |
|---------|-------------|
| Заголовки H1–H3 | `/` → выберите уровень |
| Жирный текст | `Ctrl+B` или кнопка **Ж** |
| Курсив | `Ctrl+I` |
| Список | `/` → Список или `-` в начале строки |
| Таблица | `/` → Таблица |
| Изображение | `/` → Изображение → загрузите или вставьте URL |
| Видео | `/` → Видео → вставьте ссылку на YouTube/Vimeo |
| Разделитель | `/` → Разделитель |
| Код | ` ``` ` для блока кода |

### Шаг 3. Сохраните

Документ сохраняется автоматически по мере редактирования. Закройте вкладку — данные не потеряются.

## Добавление файла

1. Нажмите **«+ Создать»** → **«Файл»**
2. Перетащите файл в область загрузки или нажмите «Выбрать файл»
3. Введите название (необязательно — автоматически берётся имя файла)
4. Нажмите **«Загрузить»**

Поддерживаемые форматы: PDF, Word, Excel, PowerPoint, ZIP, изображения и другие. Максимальный размер — 50 МБ.

---

## Добавление ссылки

1. Нажмите **«+ Создать»** → **«Ссылку»**
2. Введите URL
3. Введите название и, опционально, описание
4. Нажмите **«Сохранить»**

---

## Перемещение и переименование

Чтобы переместить материал в другую папку:
1. Наведите курсор на элемент
2. Нажмите **«⋯»** → **«Переместить»**
3. Выберите папку назначения

Для переименования: **«⋯»** → **«Переименовать»**.

---

## Настройка доступа

По умолчанию материалы видны только администраторам. Чтобы открыть доступ участникам:

1. Откройте папку или документ
2. Нажмите **«⋯»** → **«Настройки доступа»**
3. Переключите на **«Участники»** или **«Публично»**

> 💡 Если открыть доступ к папке, все вложенные материалы тоже станут доступны.
