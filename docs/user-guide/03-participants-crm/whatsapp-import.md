# Импорт истории из WhatsApp

## Зачем это нужно

Если часть общения вашего сообщества происходит в WhatsApp:

- Объедините данные из разных мессенджеров
- Получите полную картину активности участников
- Используйте историю WhatsApp для AI-анализа

---

## Необходимые условия

- Доступ к чату в WhatsApp
- Возможность экспортировать чат
- Организация создана в Orbo

---

## Экспорт из WhatsApp

### На телефоне (iOS/Android)

1. Откройте чат в WhatsApp
2. Нажмите на название чата вверху
3. Прокрутите вниз до **"Экспорт чата"**
4. Выберите **"Без медиафайлов"**
5. Сохраните файл

[СКРИНШОТ: Меню чата в WhatsApp с пунктом "Экспорт чата"]

### На компьютере (WhatsApp Web)

Экспорт через WhatsApp Web не поддерживается. Используйте мобильное приложение.

### Формат файла

Вы получите файл `.txt` с таким содержимым:

```
15.01.2024, 10:32 - Иван Петров: Привет всем!
15.01.2024, 10:33 - Мария Иванова: Привет!
15.01.2024, 10:35 - +79001234567: Доброе утро
```

## Импорт в Orbo

### Шаг 1. Откройте раздел Telegram

В левом меню перейдите в **Telegram**.

### Шаг 2. Выберите вкладку WhatsApp

Нажмите вкладку **"WhatsApp"**.

[СКРИНШОТ: Вкладки Telegram/WhatsApp]

### Шаг 3. Нажмите "Импортировать"

Найдите кнопку **"Импортировать чат WhatsApp"**.

### Шаг 4. Загрузите файл

1. Нажмите **"Выбрать файл"**
2. Выберите экспортированный `.txt` файл
3. Укажите название чата (для отображения в Orbo)

[СКРИНШОТ: Форма импорта — поле загрузки файла и поле названия]

### Шаг 5. Дождитесь обработки

Orbo обработает файл:
- Распарсит сообщения
- Создаст профили участников по номерам телефонов
- Добавит сообщения в историю

---

## Результат

### Участники

Для каждого участника WhatsApp-чата создаётся профиль:
- По номеру телефона
- Или по имени, если он сохранён в контактах

### Сообщения

Все сообщения появятся:
- В ленте активности участников
- На дашборде (учитываются в статистике)
- Доступны для AI-анализа

### Отметка источника

Сообщения из WhatsApp отмечены иконкой 📱, чтобы отличать от Telegram.

[СКРИНШОТ: Лента активности — сообщения с иконками Telegram и WhatsApp]

## Объединение с Telegram-профилями

### Автоматическое

Если номер телефона WhatsApp совпадает с Telegram — профили объединяются автоматически.

### Ручное

Если автоматически не объединилось:
1. Откройте раздел **Участники**
2. Найдите потенциальные дубликаты
3. Используйте функцию объединения

[→ Как объединить профили](./profiles.md#объединение-профилей-telegram--max)

## Возможные проблемы

### Неправильная кодировка

Если текст отображается некорректно:
- Проверьте, что файл в формате UTF-8
- Попробуйте экспортировать заново

### Не распознаются участники

- Убедитесь, что контакты сохранены с именами
- Номера без имён отобразятся как "+79001234567"

### Ошибка формата даты

Orbo поддерживает основные форматы дат WhatsApp:
- 15.01.2024, 10:32
- 15/01/2024, 10:32
- 01/15/24, 10:32 AM

Если формат не распознаётся — обратитесь в поддержку.

## История импортов

### Просмотр

Все импорты сохраняются в истории:
1. Перейдите в **Telegram → WhatsApp**
2. Посмотрите список **"История импортов"**

### Информация

Для каждого импорта видно:
- Название чата
- Дата импорта
- Количество сообщений
- Количество участников
