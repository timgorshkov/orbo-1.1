# Объединение профилей участников

## Когда это нужно

Один и тот же человек может оказаться в Orbo под несколькими профилями:

- Зарегистрировался на событие с одним email, а потом написал в Telegram под другим именем
- Состоит в Telegram-группе и в MAX-группе одновременно — два профиля по умолчанию отдельны
- Старый импорт из WhatsApp создал профиль по телефону, а потом тот же человек вступил в группу через Telegram

Объединение профилей устраняет дубли и собирает полную историю человека в одном месте.

---

## Вкладка «Дубликаты»

Вкладка **«Дубликаты»** доступна в профиле каждого участника для администраторов организации.

Она состоит из двух частей:
1. **Прикреплённые профили** — уже объединённые ранее профили
2. **Предложенные совпадения** — кандидаты для объединения, найденные автоматически

---

## Как Orbo находит совпадения

Orbo автоматически анализирует профили и предлагает кандидатов по следующим критериям:

| Критерий | Очки совпадения |
|----------|----------------|
| Совпадает Telegram ID | 70 |
| Совпадает MAX ID | 70 |
| Совпадает email | 60 |
| Совпадает телефон | 65 |
| Совпадает Telegram username | 55 |
| Совпадает MAX username | 55 |
| Похожее имя (каждое слово) | 20 |

Профили с ненулевым совпадением показываются в списке предложений, отсортированные по убыванию.

---

## Объединение двух профилей

### Шаг 1. Откройте основной профиль

Перейдите в **«Участники»** → откройте профиль, который хотите сделать **основным**. Именно он останется после объединения.

### Шаг 2. Откройте вкладку «Дубликаты»

[СКРИНШОТ: Вкладка Дубликаты — список предложенных совпадений]

### Шаг 3. Выберите дубликат

В списке предложений найдите нужный профиль:
- Проверьте имя, контакты, источник (TG/MAX)
- Обратите внимание на процент заполненности — предупреждение появится, если дубликат полнее основного

> ⚠️ После объединения основным останется тот профиль, который вы открыли. Убедитесь в правильности выбора — объединение необратимо (можно только «открепить» вручную).

### Шаг 4. Нажмите «Объединить»

Нажмите **«Объединить выбранного участника»**. После завершения Orbo покажет сводку:
- Какие поля были перенесены из дубликата
- Какие значения конфликтовали и куда сохранились

---

## Что происходит при объединении

| Данные | Результат |
|--------|-----------|
| Пустые поля основного профиля | Заполняются из дубликата |
| Telegram ID / MAX ID | Оба сохраняются в основном профиле |
| Конфликтующие значения | Сохраняются в характеристиках с пометкой |
| История сообщений | Объединяется из обоих источников |
| Теги | Объединяются |
| Регистрации на события | Объединяются |
| Членство в группах | Объединяется |

---

## Поиск дубликатов вручную

Если автоматические предложения не нашли нужный профиль — воспользуйтесь поиском:

1. На вкладке **«Дубликаты»** → поле **«Поиск по совпадениям»**
2. Введите имя, email или username — список фильтруется в реальном времени
3. Нажмите **«Обновить список»** для повторного автоматического поиска

---

## Управление прикреплёнными профилями

Уже объединённые профили отображаются в секции **«Прикреплённые профили»**:

**Открепить** — если профили были объединены ошибочно. Дубликат снова становится самостоятельным участником. Перенесённые данные (теги, группы) остаются у основного профиля.

**Синхронизировать поля** — повторно скопировать незаполненные поля из всех прикреплённых профилей в основной. Полезно если у дубликата появились новые данные после объединения.
