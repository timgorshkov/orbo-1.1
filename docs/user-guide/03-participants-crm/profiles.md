# Профили участников

## Зачем это нужно

Профиль участника — это карточка человека в вашем сообществе. Orbo автоматически создаёт профили и собирает информацию из всех подключённых источников: Telegram, MAX, событий и ручного ввода.

---

## Откуда берутся данные

| Источник | Какие данные |
|----------|--------------|
| **Telegram** | Имя, username, фото, Telegram ID |
| **MAX** | Имя, MAX username, фото, MAX ID |
| **Событие** | Email, телефон, «о себе», ответы на вопросы формы |
| **История сообщений** | Активность, категория, дата последнего сообщения |
| **AI-анализ** | Интересы, запросы, экспертиза, предложения |
| **Ручной ввод** | Заметки, теги, кастомные характеристики |
| **WhatsApp-импорт** | Имя, телефон, история сообщений |

## Просмотр профиля участника

### Шаг 1. Нажмите на участника

Кликните на строку в таблице или на имя участника.

### Шаг 2. Изучите профиль

Профиль содержит несколько блоков:

**Основная информация:**
- Фото, имя, username
- Email, телефон (если есть)
- Категория активности
- Дата присоединения

**Мессенджеры:**
- Telegram: @username и Telegram ID (ссылка для написания)
- MAX: @username и MAX ID (ссылка на профиль в max.ru)

**AI-обогащение:**
- Интересы
- Запросы (что ищет)
- Предложения (что может дать)
- Экспертиза

**Принадлежность к группам:**
- Список групп Telegram и MAX, где состоит участник
- Дата присоединения к каждой

**История активности:**
- Лента сообщений из всех мессенджеров с пометкой источника
- Участие в событиях

## Категории участников

Orbo автоматически распределяет людей по категориям на основе активности:

| Категория | Цвет | Критерии |
|-----------|------|----------|
| 🟢 **Ядро** | Зелёный | Активен регулярно, много сообщений |
| 🔵 **Новичок** | Синий | В группе менее 14 дней |
| 🟡 **Опытный** | Жёлтый | Давний участник, средняя активность |
| ⚪ **Молчун** | Серый | Нет сообщений 30+ дней |

Категория обновляется автоматически.

## Редактирование профиля

Администраторы могут дополнять профили участников.

### Что можно редактировать

| Поле | Описание |
|------|----------|
| Полное имя | Если нужно исправить |
| Email | Добавить вручную |
| Телефон | Добавить вручную |
| Фото | Загрузить вручную |
| Биография | Добавить описание |
| Теги | Метки для сегментации |

### Как редактировать

1. Откройте профиль участника
2. Нажмите кнопку **«Редактировать»** (✏️)
3. Измените нужные поля
4. Нажмите **«Сохранить»**

---

## Синхронизация фото

Если у участника нет фото или оно устарело, можно обновить его из Telegram:

1. Откройте профиль участника
2. Найдите кнопку **«Синхронизировать фото»**
3. Нажмите — фото загрузится автоматически

> 💡 Работает только при наличии публичного фото в Telegram.

---

## Объединение профилей (Telegram + MAX)

Если участник состоит в группах обоих мессенджеров, у него может быть два отдельных профиля. Объедините их, чтобы видеть полную историю в одном месте.

Подробнее: [Объединение профилей](./merge-profiles.md)

---

## Теги

Теги — произвольные метки для сегментации участников. Создавайте теги в **«Настройки»** → **«Теги»**, затем назначайте их участникам.

Примеры тегов: `спикер`, `VIP`, `наставник`, `ждёт ответа`.

Теги можно использовать как фильтр в списке участников и при создании анонсов.
