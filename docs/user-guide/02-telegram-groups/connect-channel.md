# Подключение Telegram-канала

## Зачем это нужно

Telegram-каналы — частый инструмент привлечения и взаимодействия с аудиторией. Подключив канал к Orbo, вы сможете:

- Видеть комментаторов постов в карточках участников
- Отслеживать базовую аналитику канала
- Находить пересечения аудитории канала и группы

---

## Что отслеживается

| Данные | Доступно |
|--------|----------|
| Комментарии к постам | ✅ |
| Авторы комментариев → карточки участников | ✅ |
| Метрики: новые комментаторы, активность | ✅ |
| Подписчики канала | ❌ (ограничение Telegram API) |
| Реакции к постам | ❌ (ограничение Telegram API) |

> Telegram API не позволяет получить список подписчиков канала или тех, кто оставляет реакции. Это техническое ограничение платформы.

---

## Как подключить канал

### Шаг 1. Откройте раздел «Каналы»

В левом меню найдите раздел **«Каналы»** (рядом с «Группами»).

### Шаг 2. Нажмите «Добавить канал»

Укажите один из вариантов:

| Параметр | Описание |
|----------|----------|
| Username канала | Например, `@my_channel` |
| ID канала (tg_chat_id) | Числовой идентификатор, если username отсутствует |
| Название | Будет подтянуто автоматически |

### Шаг 3. Добавьте бота в канал

Для сбора комментариев бот [@orbo_community_bot](https://t.me/orbo_community_bot) должен быть добавлен в **группу обсуждений** канала (Discussion Group).

1. Откройте настройки канала в Telegram
2. Перейдите в **«Обсуждение»** (Discussion)
3. Если группы обсуждений нет — создайте её
4. Добавьте бота в эту группу как администратора

### Шаг 4. Готово

После подключения Orbo начнёт собирать данные о комментариях. Авторы комментариев появятся как участники с пометкой источника «канал».

---

## Метрики канала

На странице канала отображается:

- Количество новых комментаторов за период
- Общая активность комментариев
- Список комментаторов с переходом в профиль участника

---

## Связь с участниками

Комментаторы канала автоматически создаются как участники в CRM вашей организации. Если тот же человек пишет и в подключённой группе, его активность объединяется в одном профиле.

---

## Ограничения

- **Подписчики канала** не доступны через API Telegram
- **Реакции** к постам не доступны через Bot API
- Данные собираются только после подключения (не ретроактивно)
