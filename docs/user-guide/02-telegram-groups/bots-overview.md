# Какие боты и для чего используются

## Зачем это нужно

Orbo использует четыре Telegram-бота для разных задач. Понимание их назначения поможет вам правильно настроить интеграцию и объяснить участникам, зачем нужен каждый бот.

---

## Четыре бота Orbo

| Бот | Username | Назначение |
|-----|----------|------------|
| **Community Bot** | [@orbo_community_bot](https://t.me/orbo_community_bot) | Сбор аналитики из групп |
| **Assist Bot** | [@orbo_assistant_bot](https://t.me/orbo_assistant_bot) | Уведомления и авторизация |
| **Event Bot** | [@orbo_event_bot](https://t.me/orbo_event_bot) | Регистрация на события (MiniApp) |
| **Start Bot** | [@orbo_start_bot](https://t.me/orbo_start_bot) | Регистрация новых организаций через Telegram |

---

## @orbo_community_bot — Основной бот

### Для чего нужен

Это главный бот Orbo. Добавляется в Telegram-группы как **администратор** для:

- Сбора сообщений и активности
- Создания профилей участников
- Отслеживания входов/выходов
- Публикации анонсов событий и сообщений
- Сбора комментариев из каналов (через группу обсуждений)

### Как использовать

1. Добавьте бота в группу
2. Назначьте администратором (права на чтение сообщений)
3. Подключите группу к организации в Orbo

### Что видит бот

| Данные | Доступ |
|--------|--------|
| Все сообщения в группе | ✅ |
| Информация об участниках | ✅ |
| Входы/выходы из группы | ✅ |
| Личные сообщения | ❌ |
| Другие группы | ❌ |

### Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и инструкции |
| `/help` | Справка по боту |

> 💡 **Важно:** Бот работает только в группах, подключённых к вашей организации в Orbo.

## @orbo_assistant_bot — Бот уведомлений

### Для чего нужен

Отправляет персональные уведомления администраторам и участникам:

- Уведомления о негативе в группах
- Алерты о неотвеченных вопросах
- Сигналы о неактивных участниках
- Коды для Telegram-авторизации
- Сообщение вашего Telegram User ID
- Автоматические напоминания о событиях (за 24 ч и за 1 ч)
- Онбординговые сообщения для новых пользователей

### Как использовать

1. Начните диалог с ботом: [@orbo_assistant_bot](https://t.me/orbo_assistant_bot)
2. Напишите `/start`
3. Бот будет присылать уведомления, если вы — получатель в правилах

### Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Показать ваш Telegram User ID |
| `/help` | Справка |

### Пример уведомления

```
🔴 Негатив в группе «Клиенты — Проект Альфа»

В чате обнаружена конфликтная дискуссия.

Серьёзность: средняя

[Открыть в Telegram] [Открыть в Orbo]
```

## @orbo_event_bot — Бот событий и заявок

### Для чего нужен

Позволяет участникам регистрироваться на события и подавать заявки прямо в Telegram через MiniApp:

- Открывает форму регистрации на событие внутри Telegram
- Открывает форму заявки (на вступление, услугу) внутри Telegram
- Автоматически подставляет данные из Telegram-профиля
- Показывает инструкции по оплате
- Отображает календарь событий организации

### Как использовать

Участники переходят по ссылке:

**Для событий:**
```
t.me/orbo_event_bot?startapp=e-{ID_события}
```

**Для заявок:**
```
t.me/orbo_event_bot?startapp=f-{slug_формы}
```

Ссылку можно:
- Скопировать из карточки события или формы заявки в Orbo
- Получить через кнопку «Поделиться» → «Telegram»

### Для организаторов

1. Создайте событие в Orbo
2. Нажмите **"Поделиться"** → **"Ссылка для Telegram"**
3. Отправьте ссылку в группу или личные сообщения

### Для участников

1. Перейдите по ссылке
2. Откроется MiniApp прямо в Telegram
3. Данные из Telegram подставятся автоматически
4. Нажмите "Зарегистрироваться"

## @orbo_start_bot — Бот регистрации организаций

### Для чего нужен

Позволяет создать новую организацию в Orbo прямо из Telegram — без перехода на сайт:

- Быстрая регистрация организации через диалог в Telegram
- Подключение первой группы сразу после создания
- Удобный онбординг для тех, кто пришёл из Telegram

### Как использовать

1. Начните диалог с ботом: [@orbo_start_bot](https://t.me/orbo_start_bot)
2. Напишите `/start`
3. Следуйте инструкциям бота для создания организации

### Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начать регистрацию организации |

## Частые вопросы

### Можно ли использовать только один бот?

Нет, каждый бот выполняет свою функцию:
- Для аналитики групп — нужен @orbo_community_bot
- Для уведомлений — нужен @orbo_assistant_bot
- Для регистрации на события в Telegram — нужен @orbo_event_bot
- Для создания организации через Telegram — нужен @orbo_start_bot

### Бот читает личные сообщения?

Нет. Боты Orbo:
- Читают только сообщения в группах, куда добавлены
- Не имеют доступа к личной переписке
- Не имеют доступа к другим группам

### Участники должны взаимодействовать с ботами?

- @orbo_community_bot — нет, работает автоматически в группе
- @orbo_assistant_bot — только администраторы для получения уведомлений
- @orbo_event_bot — только если регистрируются на событие через Telegram
- @orbo_start_bot — только при создании организации через Telegram

### Как отключить бота от группы?

Удалите @orbo_community_bot из администраторов группы. Или отключите группу в настройках Orbo.
