# Мини-приложения

## Что такое мини-приложения

Мини-приложения (Apps) — это кастомные интерактивные инструменты для вашего сообщества, работающие прямо внутри Telegram или MAX, а также на публичном портале организации.

Примеры мини-приложений:
- **Каталог участников** — участники заполняют визитку, остальные ищут нужных людей
- **Биржа запросов и предложений** — участники публикуют задачи, другие откликаются
- **База рекомендаций** — сбор и просмотр полезных книг, инструментов, контактов
- **Голосование / опросник** — структурированный сбор мнений с архивом ответов
- **Доска объявлений** — участники публикуют объявления с фото

---

## Как это работает

Каждое мини-приложение состоит из:

1. **Коллекций** — таблиц с данными (например: «Участники», «Рекомендации»)
2. **Схемы полей** — какие данные хранятся в каждой записи
3. **Интерфейса** — автоматически генерируется на основе схемы

Участники взаимодействуют с приложением через:
- **Telegram MiniApp** — открывается прямо в Telegram
- **MAX MiniApp** — открывается в MAX
- **Публичный портал** — браузерная версия на `my.orbo.ru/p/{org}/apps/{id}`

---

## Возможности для участников

| Действие | Описание |
|----------|----------|
| Добавить запись | Заполнить форму и опубликовать в коллекцию |
| Просматривать записи | Листать или искать по каталогу |
| Лайкать/голосовать | Ставить реакции на записи других |
| Комментировать | Оставлять комментарии к записям |

---

## Доступность

Мини-приложения доступны на тарифах **Pro** и **Club**.
