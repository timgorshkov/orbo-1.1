# Создание и публикация мини-приложения

## Создание приложения

### Шаг 1. Откройте раздел Apps

В левом меню нажмите **«Приложения»** (🧩).

[СКРИНШОТ: Раздел Apps — пустой каталог с кнопкой создания]

### Шаг 2. Создайте новое приложение

Нажмите **«+ Создать приложение»**. Вы можете:

**Вариант А — Создать через AI**
Опишите словами, что должно делать приложение. Например:
> «Каталог участников: имя, должность, компания, чем могу помочь, что ищу, фото»

AI сгенерирует структуру коллекции и настроит поля автоматически. Вы сможете скорректировать результат.

**Вариант Б — Создать вручную**
Укажите название приложения и добавьте коллекцию с нужными полями вручную.

### Шаг 3. Настройте поля коллекции

Для каждого поля укажите:
- **Название** — как поле называется для участника
- **Тип** — текст, число, URL, выбор из списка, изображение, логический переключатель
- **Обязательность** — обязательно к заполнению или нет

[СКРИНШОТ: Редактор схемы коллекции — список полей с типами]

### Шаг 4. Сохраните

Нажмите **«Сохранить»** — приложение создано. Теперь его можно публиковать.

## Публикация приложения

### Публикация в Telegram

1. Откройте приложение → вкладка **«Публикация»**
2. Скопируйте ссылку вида `t.me/orbo_event_bot?startapp=a-{id}`
3. Отправьте ссылку в нужную Telegram-группу или закрепите в описании

Участники откроют приложение одним нажатием прямо в Telegram.

### Публикация в MAX

1. Скопируйте ссылку для MAX (она немного отличается форматом)
2. Отправьте в MAX-группу

### Публикация на портале

Приложение автоматически появляется на публичном портале в разделе **«Приложения»** после создания. Прямая ссылка: `my.orbo.ru/p/{org}/apps/{id}`.

## Управление записями

Все записи, созданные участниками, доступны в административной панели:

1. Откройте **«Приложения»** → выберите нужное
2. Перейдите на вкладку **«Записи»**
3. Просматривайте, фильтруйте и при необходимости удаляйте записи

Администраторы также могут добавлять записи от имени участника.
