## 🚀 Быстрый старт

### Шаг 1. Регистрация

Перейдите на [my.orbo.ru](https://my.orbo.ru) и создайте аккаунт через email, Google или Яндекс.

### Шаг 2. Создание организации

Укажите название вашего сообщества. Orbo подскажет первый шаг исходя из вашей роли.

### Шаг 3. Подключение группы

Добавьте бота в вашу Telegram- или MAX-группу. Участники начнут появляться автоматически.

### Шаг 4. Создайте событие с MiniApp

Участники регистрируются прямо в Telegram или MAX. Напоминания отправляются автоматически.

### Шаг 5. Знайте своих людей

Карточки участников с историей посещений, активностью и контактами. AI-анализ интересов и переписки доступен прямо с дашборда.
