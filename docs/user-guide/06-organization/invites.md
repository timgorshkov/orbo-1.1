# Приглашения участников

## Зачем это нужно

Ссылки-приглашения позволяют предоставить доступ к организации в Orbo:

- Пригласить администраторов или модераторов
- Дать доступ к аналитике и дашборду

---

## Создание приглашения

### Шаг 1. Откройте настройки организации

Перейдите в **Настройки** → **Команда**.

### Шаг 2. Нажмите «Создать приглашение»

### Шаг 3. Настройте параметры

| Параметр | Описание |
|----------|----------|
| Тип доступа | `admin` — администратор, `member` — участник |
| Максимум использований | Сколько раз можно воспользоваться ссылкой |
| Срок действия | Когда приглашение истекает |

### Шаг 4. Скопируйте ссылку

Отправьте ссылку тому, кого хотите пригласить. Человек сможет присоединиться к организации, перейдя по ней.

---

## Управление приглашениями

На странице «Команда» видны все активные приглашения:

- Ссылка
- Количество использований
- Статус (активно / истекло)
- Кнопка для отзыва
