# MiniApp для регистрации на события

## Зачем это нужно

Telegram MiniApp позволяет участникам регистрироваться на события и оплачивать их **прямо в Telegram** — без перехода на внешний сайт. За счёт нативного опыта конверсия в регистрацию и оплату выше, чем при переходе на сторонний лендинг.

---

## Как это работает

1. Организатор создаёт событие в Orbo
2. Получает ссылку формата `t.me/orbo_event_bot?startapp=e-{id}`
3. Публикует ссылку в группе, канале, рассылке или рекламе
4. Участник нажимает ссылку → открывается форма регистрации внутри Telegram
5. Данные Telegram (имя, фамилия) подставляются автоматически
6. Участник регистрируется в пару нажатий

---

## Получение ссылки

### Из карточки события

1. Откройте событие
2. Нажмите **«Поделиться»**
3. Выберите **«Ссылка для Telegram»**
4. Скопируйте ссылку

### Формат

```
t.me/orbo_event_bot?startapp=e-{ID_события}
```

---

## Что видит участник

### Шаг 1. Переход по ссылке

В Telegram открывается бот [@orbo_event_bot](https://t.me/orbo_event_bot), затем MiniApp с информацией о событии.

### Шаг 2. Информация о событии

Участник видит:
- Название и описание
- Дату, время, формат (онлайн/офлайн)
- Обложку (если загружена)
- Стоимость (для платных)

### Шаг 3. Регистрация

- Имя и фамилия подставляются из Telegram
- Дополнительные поля (email, телефон) заполняются вручную
- Нажатие кнопки **«Зарегистрироваться»** завершает процесс

### Шаг 4. Оплата (для платных событий)

После регистрации участник видит инструкции по оплате или кнопку для перехода к оплате.

---

## Автоматические напоминания

После регистрации Orbo автоматически отправляет участнику напоминания о событии:

| Напоминание | Когда | Канал |
|------------|-------|-------|
| За 24 часа | За сутки до события | Личное сообщение от бота |
| За 1 час | За час до события | Личное сообщение от бота |
| В группу | Перед событием | Сообщение в Telegram-группу |

Напоминания повышают доходимость участников на мероприятие.

---

## Параллельная веб-регистрация

Для каждого события одновременно доступны оба способа регистрации:

| Способ | Формат ссылки | Когда использовать |
|--------|--------------|-------------------|
| MiniApp (Telegram) | `t.me/orbo_event_bot?startapp=e-{id}` | Публикация в Telegram, рекламные посты |
| Веб-страница | `my.orbo.ru/p/{org}/events/{id}` | Email, сайт, соцсети |

---

## UTM-метки

Ссылки на MiniApp совместимы с UTM-параметрами. Добавляйте метки для отслеживания каналов привлечения:

```
t.me/orbo_event_bot?startapp=e-{id}?utm_source=telegram&utm_medium=ad&utm_campaign=my_campaign
```
