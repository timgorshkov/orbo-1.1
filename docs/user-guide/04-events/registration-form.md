# Настройка формы регистрации

## Зачем это нужно

Форма регистрации определяет, какие данные вы собираете от участников события:

- Контактная информация
- Информация "о себе"
- Дополнительные вопросы

---

## Доступные поля

| Поле | Описание | Можно сделать обязательным |
|------|----------|---------------------------|
| Имя | Имя и фамилия участника | ✅ |
| Email | Электронная почта | ✅ |
| Телефон | Номер телефона | ✅ |
| Кратко о себе | Биография / О себе | ✅ |

---

## Настройка формы

### Шаг 1. Откройте событие

Перейдите в карточку нужного события.

### Шаг 2. Найдите настройки регистрации

Перейдите во вкладку **"Регистрация"** или найдите блок **"Форма регистрации"**.

### Шаг 3. Настройте поля

Для каждого поля выберите:

| Настройка | Значение |
|-----------|----------|
| Показывать | Да / Нет |
| Обязательное | Да / Нет |

[СКРИНШОТ: Настройки полей формы — переключатели]

### Шаг 4. Сохраните

Нажмите **"Сохранить"**.

---

## Примеры настроек

### Минимальная форма (только имя)

- Имя: показывать, обязательно
- Email: не показывать
- Телефон: не показывать
- О себе: не показывать

**Когда использовать:** бесплатные открытые мероприятия

### Стандартная форма (имя + email)

- Имя: показывать, обязательно
- Email: показывать, обязательно
- Телефон: не показывать
- О себе: не показывать

**Когда использовать:** вебинары с рассылкой ссылки на почту

### Полная форма (все поля)

- Имя: показывать, обязательно
- Email: показывать, обязательно
- Телефон: показывать, опционально
- О себе: показывать, обязательно

**Когда использовать:** нетворкинг-встречи, мастермайнды

## Как работает для участников

### При регистрации через сайт

1. Участник открывает страницу события
2. Нажимает **"Зарегистрироваться"**
3. Авторизуется (если не авторизован)
4. Заполняет форму
5. Получает подтверждение

### При регистрации через Telegram MiniApp

1. Участник переходит по ссылке в Telegram
2. Данные из Telegram подставляются автоматически
3. Участник дополняет недостающие поля
4. Нажимает **"Зарегистрироваться"**

## Автозаполнение

Если у участника есть профиль в Orbo, данные подставляются автоматически:

| Источник | Какие данные |
|----------|--------------|
| Telegram | Имя, фамилия |
| Профиль Orbo | Email, телефон, bio |
| Предыдущие регистрации | Все заполненные ранее |

Участник может отредактировать предзаполненные данные.

## Просмотр регистраций

### Список участников

1. Откройте событие
2. Перейдите во вкладку **"Участники"**

Вы увидите таблицу со всеми зарегистрировавшимися:

| Колонка | Описание |
|---------|----------|
| Имя | Имя участника |
| Email | Электронная почта |
| Телефон | Номер телефона |
| О себе | Краткая биография |
| Статус оплаты | Для платных событий |
| Дата регистрации | Когда зарегистрировался |

[СКРИНШОТ: Таблица участников события]

### Экспорт данных

1. Откройте вкладку "Участники"
2. Нажмите **"Экспорт"**
3. Выберите формат (CSV, Excel)
