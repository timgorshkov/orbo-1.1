# Публикация события в Telegram

## Зачем это нужно

Опубликуйте анонс события в Telegram-группы, чтобы:

- Участники группы узнали о мероприятии
- Могли зарегистрироваться по кнопке
- Получили красивый анонс с обложкой

---

## Способы шаринга

### 1. Публикация в группу через бота

Бот [@orbo_community_bot](https://t.me/orbo_community_bot) отправит анонс в выбранные группы.

### 2. Ссылка для Telegram MiniApp

Ссылка формата `t.me/orbo_event_bot?startapp=e-{id}` откроет регистрацию прямо в Telegram.

### 3. Обычная веб-ссылка

Ссылка на страницу события на сайте `my.orbo.ru`.

---

## Публикация через бота

### Шаг 1. Откройте событие

Перейдите в карточку события.

### Шаг 2. Нажмите "Поделиться"

Найдите кнопку **"Поделиться"** или **"Опубликовать"**.

[СКРИНШОТ: Карточка события с кнопкой "Поделиться"]

### Шаг 3. Выберите "Опубликовать в группу"

В выпадающем меню выберите **"Опубликовать в Telegram"**.

### Шаг 4. Выберите группы

Отметьте группы, в которые хотите отправить анонс.

Доступны только группы, подключённые к вашей организации.

[СКРИНШОТ: Список групп с чекбоксами]

### Шаг 5. Отправьте

Нажмите **"Опубликовать"**.

Бот отправит сообщение в выбранные группы.

---

## Как выглядит анонс

Бот отправляет сообщение с:

1. **Обложкой** (если загружена)
2. **Названием события**
3. **Датой и временем**
4. **Кратким описанием**
5. **Кнопкой "Зарегистрироваться"**

## Ссылка для Telegram MiniApp

### Что это

Специальная ссылка, которая открывает форму регистрации **внутри Telegram** — без перехода в браузер.

### Как получить

1. Откройте событие
2. Нажмите **"Поделиться"**
3. Выберите **"Ссылка для Telegram"**
4. Скопируйте ссылку

Формат ссылки:
```
t.me/orbo_event_bot?startapp=e-{ID_события}
```

### Как использовать

- Отправьте в личные сообщения
- Добавьте в пост канала
- Используйте в рекламе

### Для участников

1. Участник нажимает на ссылку
2. Открывается Telegram с ботом [@orbo_event_bot](https://t.me/orbo_event_bot)
3. Открывается MiniApp с формой регистрации
4. Данные из Telegram подставляются автоматически
5. Участник завершает регистрацию

[СКРИНШОТ: MiniApp в Telegram — форма регистрации]

## Обычная веб-ссылка

### Как получить

1. Откройте событие
2. Нажмите **"Поделиться"**
3. Выберите **"Скопировать ссылку"**

Формат:
```
https://my.orbo.ru/p/{org_id}/events/{event_id}
```

### Когда использовать

- Для рассылки по email
- На сайте или в соцсетях
- Когда нужна веб-страница, а не MiniApp

---

## Сравнение способов

| Способ | Плюсы | Минусы |
|--------|-------|--------|
| **Публикация в группу** | Все видят сразу, с обложкой | Только для подключённых групп |
| **Ссылка MiniApp** | Регистрация без выхода из Telegram | Только для Telegram |
| **Веб-ссылка** | Работает везде | Переход в браузер |

---

## Повторная публикация

Можно опубликовать событие несколько раз:
- В разные группы
- При обновлении информации
- Как напоминание

Каждая публикация — новое сообщение.

---

## Возможные проблемы

### Бот не может отправить сообщение

- Проверьте, что бот — администратор в группе
- Проверьте, что у бота есть право отправлять сообщения
- Убедитесь, что группа подключена к организации

### Нет кнопки публикации

- Вы должны быть администратором организации
- Событие должно быть сохранено (не черновик)

### Ссылка MiniApp не работает

- Проверьте формат ссылки
- Убедитесь, что бот [@orbo_event_bot](https://t.me/orbo_event_bot) работает
