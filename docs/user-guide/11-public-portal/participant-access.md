# Доступ для участников

## Как участник заходит на портал

Участники могут войти на публичный портал вашей организации тремя способами:

### Вариант 1. Через Telegram (рекомендуется)

1. Участник переходит по ссылке портала
2. Нажимает **«Войти через Telegram»**
3. Получает 6-значный код в Telegram от бота [@orbo_assistant_bot](https://t.me/orbo_assistant_bot)
4. Вводит код на странице — вход выполнен

> 💡 Для этого участник должен хотя бы раз запустить [@orbo_assistant_bot](https://t.me/orbo_assistant_bot) командой `/start`.

### Вариант 2. Через MAX

1. Участник переходит по ссылке портала в браузере или через MiniApp
2. Нажимает **«Войти через MAX»**
3. Авторизация происходит автоматически через MAX WebApp

### Вариант 3. По email-ссылке

1. Участник вводит email на странице входа
2. Получает письмо со ссылкой для входа
3. Нажимает ссылку — вход выполнен

---

## Что участник видит после входа

После авторизации участник получает доступ к персонализированному разделу:

- **Мои регистрации** — все события, на которые он записан
- **История оплат** — статусы платежей по событиям
- **Профиль** — его контактные данные в организации
- **Материалы** — документы и ссылки, доступные участникам
- **Мини-приложения** — интерактивные инструменты сообщества

---

## Регистрация на событие

Зарегистрироваться на событие можно без входа на портал — через публичную страницу события или через MiniApp в Telegram / MAX.

После регистрации участник получает:
- Подтверждение в Telegram или MAX (если подключён бот)
- Напоминание за 24 часа и за 1 час до события

---

## Настройка видимости разделов

Администраторы могут управлять тем, что видят участники:

В **«Настройки»** → **«Публичный портал»**:
- Включить/отключить отдельные разделы
- Настроить, требуется ли вход для просмотра материалов
- Выбрать, отображать ли каталог участников
