# Интеграция с MAX

## Что такое MAX

MAX — российский мессенджер от VK (mail.ru), набирающий популярность в бизнес- и образовательной среде. По возможностям он схож с Telegram: группы, каналы, боты и мини-приложения.

Orbo поддерживает MAX наравне с Telegram: подключайте MAX-группы, собирайте аналитику, управляйте участниками и проводите события — всё в единой панели управления.

## Бот Orbo для MAX

Чтобы Orbo мог работать с MAX-группой, в ней должен быть добавлен бот Orbo MAX Bot в роли **администратора**.

Бот доступен в каталоге ботов MAX.

---

## Профили участников из MAX

Когда участник пишет в подключённой MAX-группе, Orbo создаёт для него профиль:

- **MAX ID** — уникальный идентификатор
- **MAX username** — @ник в мессенджере (если установлен)
- **Имя** — отображаемое имя в MAX
- **Фото** — аватар из профиля MAX

Если у этого же человека есть профиль из Telegram, профили можно объединить вручную — см. [Объединение профилей](./merge-profiles.md).

---

## Параллельное ведение групп

Многие сообщества ведут группы одновременно в Telegram и MAX. Orbo позволяет видеть активность из обоих мессенджеров в одном разделе:

- Раздел **«Участники»** → все участники, независимо от источника
- Фильтр по источнику (Telegram / MAX)
- Общая лента активности с пометкой мессенджера
