# Подключение MAX-групп

## Шаг 1. Добавьте бота в MAX-группу

1. Откройте вашу MAX-группу
2. Перейдите в настройки группы → **«Администраторы»**
3. Найдите и добавьте **Orbo MAX Bot** в качестве администратора
4. Предоставьте права: читать сообщения, отправлять сообщения, управлять участниками

[СКРИНШОТ: Настройки MAX-группы — добавление бота-администратора]

---

## Шаг 2. Подключите группу к организации в Orbo

1. В левом меню нажмите **«MAX-группы»**
2. Нажмите кнопку **«Подключить группу»**
3. Введите ID или ссылку на MAX-группу
4. Нажмите **«Подключить»**

[СКРИНШОТ: Страница MAX-групп с кнопкой подключения]

После подключения Orbo начнёт получать события из группы: сообщения, входы и выходы участников.

---

## Шаг 3. Синхронизация участников

После подключения нажмите **«Синхронизировать участников»** — Orbo загрузит список всех текущих участников группы через API MAX. Это позволяет иметь полный список с именами сразу, не дожидаясь, пока каждый участник напишет сообщение.

> 💡 Синхронизацию можно повторять вручную в любое время — например, после массового вступления новых участников.

---

## Управление подключёнными группами

На странице **«MAX-группы»** отображаются все подключённые группы:

| Колонка | Описание |
|---------|----------|
| Название | Имя группы в MAX |
| Участников | Количество по данным Orbo |
| Статус | Активен / Отключён |
| Последняя активность | Когда последнее событие |

Чтобы отключить группу — нажмите **«⋯»** → **«Отключить»**. История и участники сохраняются.

---

## Возможные проблемы

**Бот не получает события**
— Убедитесь, что бот добавлен как администратор (не просто участник)
— Проверьте, что группа публичная или бот включён в белый список

**Участники показываются как «ID: 123456»**
— Нажмите **«Синхронизировать участников»**: Orbo загрузит имена через MAX API и обновит профили

**Группа не появляется после подключения**
— Обновите страницу через несколько секунд. Подключение может занимать до 30 секунд.
