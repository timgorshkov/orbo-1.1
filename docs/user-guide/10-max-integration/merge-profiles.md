# Объединение профилей Telegram и MAX

## Зачем это нужно

Когда сообщество работает в двух мессенджерах, один и тот же человек может иметь два отдельных профиля в Orbo: один из Telegram, другой из MAX. Чтобы видеть полную картину — всю историю активности, все контакты и посещение событий в одном месте, — профили нужно объединить.

После объединения:
- Один профиль содержит данные из обоих мессенджеров
- История активности объединяется
- Теги, заметки и кастомные поля сохраняются
- Регистрации на события видны полностью

---

## Как Orbo находит возможные дубликаты

Orbo анализирует профили и предлагает кандидатов для объединения по нескольким критериям:

| Критерий | Вес совпадения |
|----------|---------------|
| Совпадает MAX ID | 70 баллов |
| Совпадает Telegram ID | 70 баллов |
| Совпадает MAX username | 55 баллов |
| Совпадает Telegram username | 55 баллов |
| Совпадает email | 60 баллов |
| Совпадает телефон | 65 баллов |
| Похожее имя | 20 баллов за каждое слово |

Совпадение ≥ 20 баллов — профиль попадает в список предложений в разделе **«Дубликаты»**.

## Объединение профилей

### Шаг 1. Откройте профиль участника

Перейдите в **«Участники»** → кликните на профиль, который хотите сделать основным.

### Шаг 2. Перейдите на вкладку «Дубликаты»

На странице профиля откройте вкладку **«Дубликаты»** (доступна только администраторам организации).

[СКРИНШОТ: Вкладка «Дубликаты» с предложенными совпадениями]

### Шаг 3. Выберите дубликат для объединения

Orbo показывает список кандидатов с:
- Именем и контактами
- Мессенджером источника (TG / MAX)
- Процентом заполненности профиля
- Причинами совпадения

Нажмите на нужный профиль, чтобы выбрать его.

> ⚠️ Если дубликат полнее текущего профиля, Orbo покажет предупреждение. Основным останется тот профиль, который вы открыли — убедитесь, что выбрали правильный.

### Шаг 4. Нажмите «Объединить»

Нажмите кнопку **«Объединить выбранного участника»**.

Orbo:
1. Перенесёт все данные из дубликата в основной профиль (заполнит пустые поля)
2. Сохранит историю активности из обоих источников
3. Пометит дубликат как поглощённый — он перестанет отображаться в общем списке

---

## Что происходит с данными при объединении

| Данные | Результат |
|--------|-----------|
| Пустые поля (email, телефон, фото) | Переносятся из дубликата |
| Конфликтующие поля | Сохраняются в «Характеристиках» с пометкой |
| История активности | Объединяется |
| Теги | Объединяются |
| Регистрации на события | Объединяются |

---

## Прикреплённые профили

После объединения «поглощённый» профиль становится **прикреплённым** и отображается в разделе **«Дубликаты»** под заголовком «Прикреплённые профили».

Вы можете:
- **Открепить** — дубликат снова станет самостоятельным участником
- **Синхронизировать поля** — повторно скопировать незаполненные поля из прикреплённых профилей в основной
